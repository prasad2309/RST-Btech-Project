RST PARAMETERS CALCULATIONS: QUICK ANALYSIS

  1. Click on the “Choose file” button in the IMPORT FILE card of the input section. A file dialog will appear. Choose a .csv file of the input dataset.
  2. Click on “Upload” button.
  3. The RST parameters will then be calculated on the input dataset and the computed results will be displayed in the Statistical Analysis section.
  4. Press on the “Download” button to download an output Excel file containing the results as showcased in the Statistical Analysis Section.

RST PARAMETERS CALCULATIONS: LIVE ANALYSIS

  1. Enter the Raw link of a .csv dataset file which is uploaded on GitHub as a repository, in the first text box and click "SUBMIT" button present in front of it.
  2. Wait for 10 seconds.
  3. A line graph will then be displayed. It is refreshed automatically after a period of 10 seconds.
  4. The user can set a threshold by giving an input in the second text field and clicking on “SUBMIT” button.

MISSING ATTRIBUTE

  1. Click on the “Choose file” button in the IMPORT MISSING VALUES FILE card of the input section. A file dialog will appear.
  2. Choose a .csv file of the input dataset and click on Upload button.
  3. An output Excel file with the computed results is automatically downloaded when the user clicks on the “Upload” button.
